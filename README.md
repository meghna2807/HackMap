# HackMap
*A web platform that helps students discover opportunities based on their year - including internships, open-source programs, research programs, hackathons, and coding competitions.*

HackMap simplifies the process of finding relevant opportunties for students by filtering them by type and academic year.

## Tech Stack
- **Languages:** HTML, CSS, JavaScript
- **Hosting Platform:** GitHub

## Features
- Filter opportunties based on academic year.
- Separate categories for internships, open-source programs, research programs, hackathons, and coding competitions.
- Simple and responsive user interface.

## Future Improvements
- Add a submission form for users to share new opportunities.
- Add user authentication for saving opportunities.
- Enable email notifications to set reminders for upcoming deadlines.
